# 1 定义结构体

## 1.1 普通结构体

``` rust
struct Point {
  x: f64,
  y: f64,
  z: f64, // <- 必须分号结尾
} // <- 无分号结尾
```

这种是普通结构体

## 1.2 元组结构体

元组结构体的每个字段没有名字

``` rust
struct Point(f64, f64, f64); // <- 有分号结尾，最后一个字段没有逗号
```

它的调用同元组，用索引调用。
