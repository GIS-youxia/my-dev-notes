# field 类

- 是否是 `3dSceneLayer.json` 的顶级属性：是

## 类描述



## 举例说明

``` JSON
{
    "name": "NAME",
    "type": "esriFieldTypeString",
    "alias": "名称"
}
```

## ① [属性] name，必选

名称

## ② [属性] type，必选

值类型，可选枚举值有：

- esriFieldTypeString
- esriFieldTypeDate
- esriFieldTypeOID
- esriFieldTypeInteger
- esriFieldTypeDouble
- ...

## ③ [属性] alias，

别名
