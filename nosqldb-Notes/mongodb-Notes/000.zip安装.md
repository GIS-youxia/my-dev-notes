定义路径如下：

- 程序安装路径：D:\databases\mongo\mongodb-win32-x86_64-windows-4.4.1
- 数据目录路径：D:\databases\mongo\mongodata
- 日志路径：D:\databases\mongo\mongolog\mongod.log

- 安装为Windows服务的配置文件路径：D:\databases\mongo\mongod.cfg

以上路径均任意，请不要有中文和空格，我也不知道会有什么问题。

# 1. 下载解压

解压参考上述安装路径，也可随意指定。

记得把 `<程序安装路径>\bin` 配到环境变量下。

# 2. 先跑一次

目的是初始化，请用 cmd（管理员模式），powershell不知道有没有用。

``` SHELL
mongod.exe --dbpath=D:\databases\mongo\mongodata
```

然后能浏览器访问 `localhost:27017` 后，`Ctrl+C` 结束它。

关闭cmd窗口。

# 3. 安装为 Windows 服务

D:\databases\mongo\mongod.cfg 的内容如下：

``` YAML
systemLog:
  destination: file
  path: D:/databases/mongo/mongolog/mongod.log
storage:
  dbPath: D:/databases/mongo/mongodata
net:
  bindIp: 127.0.0.1,0.0.0.0
  port: 27017
```

大小写敏感，这个配置文件的写法参考：[配置文件写法](https://docs.mongodb.com/manual/reference/configuration-options/)

这个文件在Windows上默认路径是 `<install directory>/bin/mongod.cfg`，是一个yaml格式的（不支持tab，用空格）

官方的配置文件长这样：

![image-20201022192722520](attachments/image-20201022192722520.png)

然后执行命令（仍然是 管理员模式的 cmd 窗口执行）：

``` SHELL
mongod.exe --config D:\databases\mongo\mongod.cfg --install
```

应该是没有提示的，然后就可以执行命令启动服务了：

``` SHELL
net start MongoDB
```

# 几个二进制文件是什么

- mongod：数据库系统的主进程，提供监听和服务等
- mongo：类似于客户端的程序，用于连接mongodb数据库服务，应该是shell吧
