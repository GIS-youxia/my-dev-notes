# 使用 WGSL

wgsl，即 WebGPU Shader Language，专为 WebGPU 设计

## 为什么不用 glsl 转 spir-v ？

因为终究会废弃，虽然 wgsl 更新非常频繁，但是一路一点点学习，有助于成长式学习。

# 技术前提

- ECMAScript 标准中的 async / await 的语法糖用法或 Promise 用法。
- DOM 基本知识。
- 有一定的 WebGL 基础。
